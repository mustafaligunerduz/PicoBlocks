# PicoBlocks

PicoBlocks is a sandbox building game inspired by block-based games.

## Features

- Place and break blocks
- Multiplayer support
- Inventory system
- Custom skins
- Simple world generation

## Controls

- WASD = Move
- Space = Jump
- Left Click = Break Block
- Right Click = Place Block

## Play

Open the game in your browser or visit the GitHub Pages version.
